# 🦞 QClaw 统一指挥中心

> QClaw 多渠道会话统一管理平台

## 功能特性

- 📊 **数据概览** - 招链通业务数据统一展示
- 🚀 **快捷入口** - 一键跳转到各业务系统
- 📋 **任务追踪** - 跨会话任务统一管理
- 🔗 **会话中枢** - 协调微信、飞书等多渠道对话

## 快速部署

### 1. 创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称：`qclaw-command-center`
3. 选择 **Public**
4. 点击 **Create repository**

### 2. 上传代码

在仓库页面点击 **Add file** → **Upload files**，把本文件夹下的所有文件拖进去

### 3. 部署到 Vercel

1. 访问 https://vercel.com
2. 点击 **Add New** → **Project**
3. 选择 **Import Git Repository**
4. 选择刚创建的 `qclaw-command-center` 仓库
5. 点击 **Deploy**

### 4. 完成！🎉

Vercel 会自动生成一个免费域名，访问即可使用

## 自定义域名（可选）

1. 在 Vercel 项目设置中添加域名
2. 在域名服务商添加 DNS 记录
